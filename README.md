## Customization

### Update Personal Information

1. **Name & Title**: Edit the hero section in `index.html`
2. **About Section**: Update the about text and statistics
3. **Skills**: Modify the skills grid with your own skills
4. **Portfolio**: Add your own projects with images and links
5. **Contact**: Update email, phone, and social media links

### Change Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... */
}
```

### Add Your Images

Replace the Unsplash image URLs with your own images:
- Hero image
- About image
- Portfolio project images
